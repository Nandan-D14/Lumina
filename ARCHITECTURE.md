## Module Responsibilities

| Module | Responsibility |
|--------|----------------|
| **Frontend** | React-based UI with insights dashboard, chart controls, and history sidebar |
| **API Layer** | FastAPI endpoints for /health, /analyze, /export, /legacy operations |
| **Orchestrator** | Central coordination of multi-agent workflows with state management |
| **Agents** | Specialized AI agents for research, analysis, visualization, and response assembly |
| **Services** | Business logic including artifact storage, export, and web interactions |
| **Tools** | Utility functions for data parsing, web scraping, and searching |
